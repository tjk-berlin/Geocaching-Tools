# Geocaching-Tools
This repository contains a collection of Python tools designed to help create, generate, and customize geocaching puzzles. The scripts support a variety of puzzle types commonly used in geocaching â€” from cipher helpers and coordinate transformations to automated clue generation and utility functions that simplify the puzzle-creation workflow.

Whether you're building puzzles for your own caches or developing tools for the geocaching community, this repo provides a flexible and extensible foundation.

## Features
- Utilities for generating and solving common geocaching ciphers
- Coordinate manipulation and conversion helpers
- Puzzle-generation scripts for creating clues, patterns, and encoded messages
- Easy-to-extend structure for adding new puzzle tools
- Lightweight and fully written in Python

## Use Cases
- Creating custom geocaching riddles and mystery caches
- Automating parts of the puzzle-design process
- Building repeatable puzzle patterns or helpers
